01

대상 사용자

  • 작업자
  • 현장 리더
  • 교육·표준작업 담당자
02

입력

  • 공정 단계
  • 공구 정보
  • 이미지 또는 영상 자료
  • 단계별 확인 규칙
03

운영 이벤트

  • 단계 시작
  • 확인 필요
  • 작업 완료
  • 예외 보고
04

결과

  • 작업자 시점의 단계별 안내
  • 필요한 공구와 확인 항목
  • 단계 확인 기록
05

파일럿 적용 조건

  • 안내를 볼 기기와 작업 위치 정의
  • 표준작업 콘텐츠 책임자 지정
  • 변경 승인과 업데이트 절차 정의
06

측정 KPI

  • 신규 작업자 교육 시간
  • 누락 단계 수
  • 표준 준수율
  • 작업 중 문의 횟수
