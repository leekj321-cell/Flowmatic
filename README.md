## v5.3 상세 페이지 구조
각 기술 상세 페이지를 두 개의 주요 섹션으로 압축했습니다.
1. 제품 메시지 + 작동 애니메이션 + 핵심 특징
2. 제품 데모 + 문의 영역

제목 줄바꿈은 수동 줄바꿈을 줄이고, 가능한 한 문장 단위로 구성했습니다.


## v5.4 typography rules
- Major headlines use deliberate sentence/clause line breaks.
- Each line is kept intact with automatic font-size reduction when needed.
- Korean and English have independent copy and line structure.


## v5.5 typography
- 문장 단위 줄바꿈은 유지합니다.
- 각 문장은 해당 셀 폭에 들어가는 가장 큰 글자 크기로 자동 확대됩니다.
- KR/EN 전환, 브라우저 크기 변경, 폰트 로딩 후 자동으로 다시 계산합니다.
